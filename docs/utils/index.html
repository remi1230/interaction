replaceAvOnEllipse

Fait tourner tous les avatars autour d'une ellipse en fonction de la variation de la valeur du contrôle.

replaceAvOnEllipse(ctrl: Object, avatars: Array<Object>, canvas: Object, rad: number)
Parameters
ctrl (Object) L'objet de contrôle contenant les valeurs de rotation.
Name Description
ctrl.last_vals Array<number> Tableau des valeurs précédentes du contrôle.
ctrl.value number La valeur actuelle du contrôle.
avatars (Array<Object>) Tableau des objets avatar à faire tourner.
canvas (Object) L'objet canvas fournissant le point central.
rad (number) Le facteur de conversion des degrés en radians.

Chaque objet avatar doit avoir :

  • {Function} rotateCalc - Fonction pour calculer la position après rotation.
  • {Object} [center] - Centre optionnel pour la rotation.
  • {number} x - La coordonnée x (sera mise à jour).
  • {number} y - La coordonnée y (sera mise à jour).

replacesInFormuleColor

Remplace les éléments symboliques d'une formule de couleur par leurs équivalents JavaScript utilisables dans le contexte de l'objet avatar. Par exemple, remplace 'H', 'S', 'L', 'A' par 'h', 's', 'l', 'a', et d'autres symboles comme 'D', 'sz', 'cx', 'cy', 'vx', 'vy', 'v', 'x', 'y' par des expressions ou propriétés JavaScript correspondantes. Utile pour permettre l'évaluation dynamique de formules de couleur personnalisées.

replacesInFormuleColor(colorElem: string): string
Parameters
colorElem (string) La formule de couleur à transformer.
Returns
string: La formule transformée, prête à être évaluée en JavaScript.

testFormule(formule: string): number
Parameters
formule (string) La formule mathématique à tester (ex: "2*x+y").
Returns
number: Le résultat de l'évaluation de la formule, ou NaN si l'évaluation échoue.

getRandomPoint

Parameters
coeff (number) Un coefficient entre 0 et 1 qui contrôle la dispersion du point aléatoire. Une valeur proche de 1 permet des points partout sur le canvas, tandis qu'une valeur proche de 0 restreint les points vers le centre.
